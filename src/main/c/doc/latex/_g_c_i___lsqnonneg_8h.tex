This function solves the non-\/negative least squares problem. 

Minimise $\vert$\-Ax-\/b$\vert$ subject to x $>$= 0 (where $\vert$v$\vert$ is the 2-\/norm of the vector v). !!! N\-B\-: A and B will both be modified unless preserve is non-\/zero (see below).
